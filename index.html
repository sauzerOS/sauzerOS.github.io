From scratch source distro and package manager (hokuto)
Can be bootstrapped from any distro with basic development tools
Based on Arch Linux and LFS/BLFS/MLFS/GLFS

Main Features:
- Systemd
- Kernel with cachyos patches
- Xlibre
- Kde plasma
- Xfce
- Multilib support for gaming
- Binary repo support (S3 or R2)
