Bootstrap Process

Bootstrap process:

  • get latest hokuto release
    wget hokuto.xdd.ovh/hokuto.tar.xz
    tar xvf hokuto.tar.xz && mv hokuto-* hokuto
  • build bootstrap chroot
    hokuto bootstrap <target dir>
  • chroot in bootstrap chroot
    hokuto chroot <target dir>
  • configure chroot
    hokuto settings (generate and set signing key, set build type)
  • bootstrap python and python build tools
    hokuto build python
    pip install build flit_core installer
    hokuto build --nodeps python-build python-flit-core python-installer python-packaging python-pyproject-hooks python-setuptools
    hokuto alt discard-unmanaged
  • build clean toolchain
    hokuto build --rebuilds toolchain
  • build base system
    hokuto build --rebuilds base-system
